# Tymur-Turchak-Portfolio
This repository contains the source code for my personal portfolio website, designed to showcase my qualifications and technical skills as an applicant for an Ausbildung as an Fachinformatiker in Germany.
Purpose
The goal of this project is to provide a comprehensive overview of my profile for potential employers, demonstrating both my professional readiness and my foundational programming abilities.
🛠 Features
Profile Overview: Detailed section on my Hard Skills (Tech Stack) and Soft Skills.
Language Proficiency: Clear visualization of my language levels (German, English, etc.).
Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)
